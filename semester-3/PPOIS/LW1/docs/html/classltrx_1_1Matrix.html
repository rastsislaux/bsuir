Detailed Description

template<typename T>
class ltrx::Matrix< T >

Matrix class.

Class that represents matrix of any numeric type, supports some features of functional-style programming, all of basic math operators.

Template Parameters
T- type of matrix elements
Warning
supports only numeric types (int, long, float, double, etc.)
Author
R. Lipski
